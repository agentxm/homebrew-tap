# AgentXM Homebrew Tap

Official Homebrew tap for [AgentXM](https://github.com/agentxm) CLI tools.

## Installation

```bash
brew tap agentxm/tap
brew install axm
```

Or install directly:

```bash
brew install agentxm/tap/axm
```

## Available Formulas

| Formula | Description                                                |
| ------- | ---------------------------------------------------------- |
| `axm`   | The official CLI for AgentXM - Agent Experience Management |

## Updating

```bash
brew update
brew upgrade axm
```

## Troubleshooting

If you encounter issues:

```bash
brew untap agentxm/tap
brew tap agentxm/tap
brew install axm
```
